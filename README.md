PermaProg - permanent progression and roguelite elements for Slay the Spire 2

This mod introduces a new roguelite system. All gold earned during the run is also converted into the new 'Currency'. Currency is available to spend freely in the PermaProg Shop for upgrades that affect the run, be it more starting gold, more max HP or other exciting upgrades to discover and unlock.
The mod UI is accessible via a new 'enter shop'-button in the character selection screen.
